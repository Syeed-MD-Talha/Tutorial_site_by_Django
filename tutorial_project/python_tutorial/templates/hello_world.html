Let's Get Started!

Step 1: Create Your Python Workspace

  1. Create a new folder called helloworld anywhere on your computer.
  2. Open Visual Studio Code (VS Code) and select this folder.
  3. In VS Code, create a new file and name it app.py.

Step 2: Write Your First Python Code

In your app.py file, type this magical line:

print('Hello, World!')

That's it! You've just written your first Python program. ðŸŽ‰

What's happening here?

print() is like a friendly robot in Python that takes whatever you give it and shows it on the screen. Here, we're asking it to display 'Hello, World!'.

What is a Function?

Think of a function like a kitchen appliance. You put something in (input), it does something (process), and gives you something back (output).

  • A toaster takes bread (input), heats it (process), and gives you toast (output).
  • The print() function takes your message (input), prepares it (process), and displays it on the screen (output).

Running Your Python Program

Now, let's see your creation come to life!

  1. Open the Terminal in VS Code (Terminal > New Terminal).
  2. Type this command and press Enter:
python app.py

If you're using Mac or Linux, you might need to use python3 instead:

python3 app.py

You should see your message appear like magic:

Hello, World!

Congratulations! You've just run your first Python program! ðŸŽŠ

Extra Fun: Python IDLE

Python comes with a built-in playground called IDLE. It's like a sandbox where you can type Python code and see the results immediately. Give it a try:

  1. Open Python IDLE on your computer.
  2. Type print('Hello, Python!') and press Enter.
  3. Watch your message appear instantly!
